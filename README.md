# 📊 Lenta SQL & Forecasting Analysis

## 📌 Описание проекта

Проект направлен на решение аналитических задач с использованием SQL и Excel, включая расчёт ключевых метрик продаж, анализ ассортимента и оценку качества прогнозирования.

В рамках проекта выполнены следующие задачи:

- расчёт средней цены товаров по магазинам  
- анализ продаж по датам  
- ранжирование товаров по выручке  
- сравнение прогноза и фактических продаж  
- оценка точности прогнозирования (WAPE, отклонения)

## 🗂 Структура данных для SQL

В проекте используются следующие таблицы:

### 🧾 Journals (Журнал продаж)

- `JournalID` — идентификатор чека  
- `Date` — дата продажи  
- `Shop` — магазин  
- `TotalQty` — общее количество товаров  
- `TotalAmount` — сумма продаж  

### 📦 Lines (Строки продаж)

- `JournalID` — идентификатор чека  
- `Itemid` — товар  
- `Qty` — количество  
- `Amount` — сумма  

### 🏷 Goods (Справочник товаров)

- `Itemid` — идентификатор товара  
- `Itemname` — название товара  

### 📈 Forecast (Прогноз продаж)

- `Date` — дата  
- `Itemid` — товар  
- `Shop` — магазин  
- `fcst` — прогноз количества

## 🧠 SQL-запросы

### 1. Средняя цена товара по магазину

```sql
SELECT Itemid,
       1.0 * SUM(Amount) / SUM(Qty) as avg_price
FROM Lines as l
JOIN Journals as j on j.JournalID = l.JournalID
WHERE Shop = 943
GROUP BY Itemid;
```

---

### 2. Количество продаж по товарам за дату

```sql
SELECT Itemid, 
       SUM(Qty)
FROM Lines as l
JOIN Journals as j on j.JournalID = l.JournalID
WHERE Date = '2011-01-01'
GROUP BY Itemid;
```

---

### 3. Рейтинг товаров по выручке

```sql
SELECT itemname,
       l.Itemid as Itemid,
       SUM(Amount) as total_sum,
       DENSE_RANK() OVER(order by SUM(Amount) desc) as rnk
FROM Lines as l 
JOIN Goods as g on g.Itemid = l.Itemid
GROUP BY itemname, l.Itemid;
```

---

### 4. Сравнение прогноза и факта

```sql
SELECT l.Itemid as Itemid, 
       itemname, 
       MAX(fcst) as prognoz, 
       SUM(Qty) as fact
FROM Lines as l 
JOIN Journals as j on j.JournalID = l.JournalID
JOIN Goods as g on g.Itemid = l.Itemid
LEFT JOIN Forecast as f 
    on f.Itemid = l.Itemid 
   AND f.Shop = j.Shop 
   AND f.Date = j.Date
WHERE j.Shop = 1543 
  AND j.Date = '2011-01-02'
GROUP BY l.Itemid, itemname;
```
